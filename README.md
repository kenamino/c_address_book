# C语言命令行通讯录管理系统 (C Command-Line Address Book Manager)

## 项目简介

这是一个基于C语言实现的**命令行通讯录管理系统**,旨在帮助初学者掌握C语言的基础知识，包括**结构体**、**函数模块化**以及**文件操作**，同时体验完整的项目开发流程。

本项目实现了联系人的增、查、删以及数据的持久化存储功能.

## 主要功能

| 序号 | 功能名称 | 核心技术点 | 备注 |
| :---: | :--- | :--- | :--- |
| 1 | **添加联系人** | 结构体、`fgets`安全输入 | 录入姓名、电话、邮箱 |
| 2 | **查看所有联系人** | 数组遍历 | 显示所有已存储的联系人信息 |
| 3 | **搜索联系人** | `strstr`模糊匹配 | 支持按姓名或电话进行模糊查询 |
| 4 | **删除联系人** | 数组元素移动 | 通过ID删除指定联系人，并维护数据连续性 |
| 5 | **数据持久化** | 二进制文件I/O (`fread`/`fwrite`) | 退出时自动保存，启动时自动加载数据 |

## 技术栈

*   **语言:** C
*   **数据结构:** 结构体、数组
*   **核心库:** `stdio.h`, `stdlib.h`, `string.h`
*   **数据存储:** 二进制文件 (`contacts.dat`)

## 如何运行

### 1. 环境准备

需要安装C语言编译器，例如 `GCC`。
在大多数Linux系统或安装了MinGW/Cygwin的Windows系统上，都可以使用以下命令检查：

```bash
gcc --version
```

### 2. 编译项目

进入项目根目录，使用 `gcc` 编译 `main.c` 文件：

```bash
# 编译源代码，生成名为 address_book 的可执行文件
gcc main.c -o address_book
```

### 3. 运行程序

编译成功后，运行生成的可执行文件：

```bash
# 在 Linux/macOS 上运行
./address_book

# 在 Windows 上运行
address_book.exe
```

### 4. 交互示例

程序启动后，将显示主菜单：

```
========== 命令行通讯录管理系统 ==========
1. 添加联系人
2. 查看所有联系人
3. 搜索联系人
4. 删除联系人
5. 保存并退出
==========================================
请输入您的选择: 
```
输入对应的数字即可执行相应操作。
